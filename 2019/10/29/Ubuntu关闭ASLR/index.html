Ubuntu关闭ASLR

Apr1L's blog
2019-10-29
Linux
ASLR, Linux, Ubuntu

确认ASLR是否已经被打开，“2”表示已经打开

suv11@ubuntu:~$ cat /proc/sys/kernel/randomize_va_space 
2

直接使用sudo会出现下面结果：

suv11@ubuntu:~$ sudo echo 0 > /proc/sys/kernel/randomize_va_space
bash: /proc/sys/kernel/randomize_va_space: 权限不够

切换至root下关闭ASLR：

suv11@ubuntu:~$ su
密码： 
root@ubuntu:/home/suv11# echo 0 > /proc/sys/kernel/randomize_va_space
root@ubuntu:/home/suv11# cat /proc/sys/kernel/randomize_va_space
0
root@ubuntu:/home/suv11# su suv11
suv11@ubuntu:~$
