easywrite

之后复现了一下easywrite，主要参考wp

程序逻辑很简单：

  1. 申请1个0x300的堆块A，写0x2ff字节；
  2. 输入一个地址，把该地址改写为A的地址
  3. 申请一个0x30的堆块B，写0x2f字节
  4. free(B)

所以往哪里写A的地址就很关键，通过分析一波malloc.c源码：

static __thread bool tcache_shutting_down = false;
static __thread tcache_perthread_struct *tcache = NULL;
